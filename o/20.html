5. Чтение и вывод в строки в языке C при помощи функций sprintf и sscanf

В языке C есть функции sscanf и sprintf, позволяющие осуществлять форматный ввод-вывод с использованием строк. Например, пусть строка s содержит два целых числа, записанных через пробел и необходимо извлечь значения этих чисел и записать их в переменные a и b. Это делается при помощи функции sscanf, у которой первый параметр - строка, из которой читаются данные (строка языка C, то есть массив символов или указатель типа char *), а остальные параметры - как у функции scanf, то есть форматная строка и адреса переменных для записи данных. Пример:
 

sscanf(s, "%d%d", &a, &b);

Функция sscanf реализует все возможности чтения данных, которые есть в функция scanf и fscanf.
Аналогично для вывода данных не в файл, а в строку, используется функция sprintf. Её первый параметр - строка, остальные параметры - как для функции printf. Пример:
 

sprintf(s, "%d %d", a, b);


Эти функции объявлены в заголовочном файле stdio.h.
